## Development

Environmental requirements

- node >= v16.18.0 LTS
- npm >= 8.19.2
- yarn >= 1.22.19

```bash
# clone project
git clone https://github.com/open-job/openjob-ui.git
# Enter the project directory
cd openjob-ui
# install nodemodel(Must use yarn)
yarn
# run dev
npm run dev

```
